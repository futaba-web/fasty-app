# Fasty（ファスティング × 瞑想の習慣化アプリ）

> **キャッチコピー**：**「身体を整えるだけじゃなく、心も整えるファスティング」**  
> ※ 本READMEは卒業制作の企画提出用です（メンタルケア寄りに改訂）

---

## サービス概要（3行）
- ファスティングと**瞑想/呼吸**を組み合わせ、空腹の波・ストレスを“やり過ごす力”を育てる習慣化アプリ。  
- 断食は**保存時に自動判定**、瞑想は**5/10/20/30分の短尺ガイド**へワンタップ導線（MVPは外部リンク）。  
- **ミニマルUI × ポジティブなフィードバック**＋**終了時コメント（1行）**で、無理なく毎日つづく。

---

## 用語について

- 本アプリはファスティング＝時間制限食（Time-Restricted Eating / IF）の
- 開始・終了時刻を記録し、継続を後押しするツールです。
- ここでいうファスティングは いつ食べるか の時間窓管理を指し、医療上の絶食（NPO）や長期の完全断食は対象外です。

- 本プロダクトの立場（推奨ポリシー）

- Fasty はウェルネス文脈におけるファスティング習慣の実践を推奨します。
- ただし、効果・安全性は体調や既往歴、服薬状況で異なるため、医療的判断や個別指示は行いません。

- 推奨するのは 無理のない時間窓の管理（例：12–14h から始め、生活に合わせて調整）

- 体験設計は 入力の摩擦最小化 と ポジティブな継続支援（達成/未達時のメッセージ、終了時コメント）に重点

- 医療的な絶食や長期断食の指導は行いません

## このサービスへの思い・作りたい理由
- 忙しさやストレスで「空腹の波」に飲まれ継続が難しい。入力や達成判定の手間も負担。  
- そこで、**記録は最小限（開始/終了/コメント）**に、**判定は自動化**。気分調整は**短い瞑想ガイド**と、保存時の**ひとことコメント**で支える。  
- うまくいかない日も**自己効力感を下げない**文言・設計で、翌日に橋をかける。

## 個人的な原体験と気づき
- 日常的に16時間のファスティングを取り入れつつ、最長10日間のファスティングにも挑戦（ビタミン等を含む飲料のみ）。
- ダイエット目的というよりデトックスの実験として取り組み、結果として 5kg 減の副次効果も得られました。

- 体重以外の変化が大きく、体調の安定／睡眠の質向上（短時間でも回復）／肌荒れの改善を実感。
- 「食べ物への感謝」が自然と高まり、食との向き合い方も変わりました。

- また、2年前から寝る前の瞑想を習慣化。ガイド付き音声（YouTube）を聴きながら横になって実践するだけで、
- 精神面の安定・自己肯定感の向上・睡眠の質向上を強く感じました。

- これらの体験から、断食と瞑想は相性が良いと確信。空腹の波やストレスを“やり過ごす力”をつくるうえで、
- 「入力の摩擦が少なく」「すぐ使える瞑想導線」を備えたアプリがあると継続しやすいと感じ、本サービスを企画しました。

## ユーザー層（ペルソナ）
- **忙しくてストレスが多い社会人**：短時間で気持ちをリセット（3〜10分の呼吸/瞑想）。  
- **睡眠の質を上げたい人**：夜の軽い瞑想とファスティングでリズムを整えたい。  
- **マインドフルネス/瞑想に興味がある人**：取っ掛かりが欲しい → ワンタップで開始。  
- **“ダイエットだけじゃない”人**：体重以外（気分の記録、ストリーク）で継続モチベを維持。

---

## サービスの利用イメージ
1. 目標（12/14/16/18/20/22/24h）を選んで**開始**。  
2. ファスティング中に空腹の波や集中力低下を感じたら、**5/10/20/30分の瞑想リンク**へワンタップ。  
3. **終了**→保存で**達成/失敗は自動判定**。**コメント欄**にその時の気持ちを一言。  
4. マイページで**ストリーク（断食）**や週/月の傾向を確認。

## 差別化ポイント
- **“摩擦ゼロ”の断食記録**：ワンタップ開始/終了＋**保存時の自動判定（秒精度・30s猶予）**。編集で**既存 end_time を上書きしない**。  
- **メンタル介入の同梱**：断食中に**短尺の瞑想/呼吸ガイド**へ即アクセス。  
- **ポジティブ・フラッシュ＋終了時コメント**：達成→称賛、未達→労い＋再開の背中押し。  
- **国内の“ゆる断食”最適化**：プリセット目標（12〜24h）とやさしい日本語UI。広告トラッキングなし／データエクスポート可。  
- **ミニマルUI**：入力は「開始・終了・コメント」に絞る。

## MVPの本質とスコープ
**本質**：ファスティングの継続率を上げる **摩擦最小＋前向きリスタート**  
- 入力・判断の負荷を最小化（ワンタップ開始/終了、保存時の自動判定）  
- 未達でも閉じない体験（結果別ポジティブ・フラッシュ、ストリークで小さな成功を可視化）

**MVP（必須）**
- ファスティング：開始/終了、**保存時自動判定（秒精度・30s猶予）**、**終了時コメント（1行）**  
- 画面：一覧/詳細/編集、**結果別ポジティブ・フラッシュ**  
- 指標：**ストリーク（ファスティング）** ※瞑想は将来拡張

**MVP（任意・軽量）**
- 瞑想ガイド：**外部リンク一覧（YouTube）**のみ（埋め込み/自前音声は後続）

- **ペルソナ**：忙しくてストレスが多い社会人／睡眠の質を上げたい人／ダイエットだけでなく、マインドフルネス・瞑想に関心がある人
- **検索意図**：  
  - 断食の実務：「ファスティング 記録」「断食 タイマー」「16時間断食 継続」  
  - メンタル寄り：「瞑想 ガイド 初心者」「寝る前 瞑想 音声」「マインドフルネス 5分」
- **ゴール**：LP → アカウント作成 → **初回の「開始」実行**（PWAのホーム追加は任意）

### 2. 主要チャネル（MVPで低コスト検証できる順）
1. **コンテンツSEO（LP/ブログ）**
   - キーワード例：  
     「ファスティング 記録/カレンダー/やり方/失敗」「瞑想 ガイド 5分/寝る前」  
   - 週1本：始め方・よくある失敗と対策・“挫折しない導線”の紹介。  
   - 目的：検索流入から**初回開始**までのCVR最大化（LP内にワンタップ動線）。

2. **SNS（X/Instagram）**
   - 機能：**日次の達成カード**（テキスト＋OG画像）をシェアできる導線。  
   - 投稿テンプレ・ハッシュタグ：#ファスティング記録 #今日の瞑想 #16時間ファスティング  
   - 目的：オーガニック拡散と信頼（UGC）。

3. **コミュニティ配布**
   - ヘルスケア系Discord/Slack、知人ネットワーク／プログラミングスクールでβ配布。  
   - 初期ユーザーへのオンボーディングを丁寧に行い、**質の高いフィードバック**を収集。

4. **紹介（リファラル）※軽量**
   - 招待リンク経由で初回開始すると**小さなバッジ**付与（実装は後続でも方針明記）。

> ※広告はMVPでは実施しない。まずは**有機的な獲得**でメッセージ/体験の適合を検証。

### 3. 検証順序（0→1 スプリント例）
- **Week1**：LP公開／ブログ2本／Xで告知  
  - KPI：LP→登録 **CVR ≥ 20%**、登録数 **≥ 50**  
- **Week2**：達成カードのシェア導線を追加  
  - KPI：シェア率 **≥ 5%**、流入の再帰性（自己拡散）確認  
- **Week3**：コミュニティ配布＆ユーザーインタビュー  
  - KPI：**D7継続 ≥ 30%**、NPS/定性改善案の収集

### 4. 計測と改善
- **主要KPI**：登録数、初回開始率、LP→開始のCVR、D1/D7継続、シェア率、招待リンク経由比率  
- **ツール**：GA4 or Plausible、PostHog（イベント：start/finish/share/install）  
- **改善サイクル**：週次で仮説→施策→計測→振り返り。CVRと継続に寄与しない施策は停止。

### 5. 差別化との接続
- 「**ファスティング×瞑想**」の訴求をLP＆記事で一貫表現（キーメッセージ：**“身体も心も整える”**）。  
- 初回体験は**操作最小・自動判定**でハードルを下げ、達成カードで**ポジティブな継続**を促す。

## 健康と安全について（重要）
ファスティングや食事制限は体調や既往歴、服薬状況によって**リスクが生じる**ことがあります。実施可否は**必ず医療専門職に相談**してください。本アプリは医療行為や指示を提供しません。また、ファスティング中は適度な運動も心がけてください。

### 想定されるデメリット/副作用(例)
- 空腹感、倦怠感、頭痛、いら立ち、集中力低下、便秘、睡眠の質の低下 など。多くは一過性ですが、続く/強い場合は中止してください。

- 糖尿病などの持病がある方、低血糖を起こしやすい方、服薬に食事が必須の方は、低血糖・電解質異常などのリスクに注意。

- 長時間ファスティングを慢性的に繰り返す、急速な減量といった極端な実践は胆石や体調悪化のリスクがあります。

- 高齢者や低栄養が疑われる状態では、体力低下や転倒リスクに留意が必要です。
### 実施を避ける/医師に必ず相談すべき**ケース(例)
- 妊娠中/授乳中、18歳未満、摂食障害の既往/症状がある方

- 糖尿病・心腎疾患などの慢性疾患がある方、服薬に食事が必須の方 など

## 中止・受診を検討すべきサイン
- 強いめまい・失神しそうな感覚、胸痛・動悸、嘔吐や下痢が続く、著しい脱水や混乱など。安全第一で行ってください。
> **免責事項**：本アプリは健康や疾病の診断・治療・予防を目的としたものではありません。記載内容は一般情報であり、医療アドバイスではありません。体調不良時は直ちに中止し、医療機関に相談してください。
